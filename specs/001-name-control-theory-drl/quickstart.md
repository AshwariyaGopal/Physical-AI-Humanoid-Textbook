# Quickstart

To preview the generated content locally:

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Docusaurus Server**:
    ```bash
    npm run start
    ```

3.  **Navigate**:
    Open `http://localhost:3000/docs/Module-5-Control-Theory-for-Humanoids/Week-7-DRL-Dexterity`
